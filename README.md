# Contra Violência Patrimonial

Landing page informativa e responsiva dedicada à conscientização sobre a violência patrimonial contra a mulher.

## 🚀 Tecnologias
- HTML5 / JavaScript (Vanilla)
- **Tailwind CSS**: Estilização moderna e responsiva.
- **AOS (Animate On Scroll)**: Animações fluidas durante a navegação.
- **Google Fonts**: Tipografia Inter e Poppins.

## 💻 Como Visualizar
1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador moderno.

## 📝 Sobre o Projeto
Este projeto visa informar mulheres sobre os sinais da violência patrimonial, as leis de proteção (como a Lei Maria da Penha) e o panorama do sistema de justiça brasileiro, utilizando uma interface moderna e acessível.
